Dataset Information

This sentiment analysis model was trained on the Thread app reviews dataset from Kaggle.

Dataset Source:

https://www.kaggle.com/datasets/saloni1712/threads-an-instagram-app-reviews

Dataset Details:

  • 32,910 reviews from Google Play Store and Apple App Store
  • Contains review text, ratings, and dates
  • Reviews from both Android and iOS users
  • Used for training BERT-based sentiment classification model

Model Architecture:

  • LSTM (Long Short-Term Memory) with Bidirectional layers
  • Fine-tuned for sentiment analysis (Positive, Negative, Neutral)
  • Optimized for text classification tasks
